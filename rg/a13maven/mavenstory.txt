mvn archetype:generate -DgroupId=com.example.addition -DartifactId=AddTwoNumbers -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false


cd AddTwoNumbers


know the folder structure, then  write the Java Program

mvn compile


open pom.xml  add this if you want your jar file where
it is executable.

<build>
    <plugins>
        <plugin>
            <groupId>org.apache.maven.plugins</groupId>
            <artifactId>maven-jar-plugin</artifactId>
            <version>3.2.2</version>
            <configuration>
                <archive>
                    <manifest>
                        <mainClass>com.example.addition.App</mainClass>
                    </manifest>
                </archive>
            </configuration>
        </plugin>
    </plugins>
</build>


package the code
mvn package


running the jar file
java -jar target/AddTwoNumbers-1.0-SNAPSHOT.jar

unit testing the code
ensrue test code is present in the src/test folder.
ensure junit dependancy in the pom.xml file.

mvn test..
